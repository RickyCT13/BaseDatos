Práctica 8.2 - Funciones en MySQL

Cierre: lunes, 24 de abril de 2023, 08:30
Práctica 8.2 - Funciones en MySQL

La Base de Datos de trabajo es Bancos.sql 

Actividad 1. SaldoTotal

Crear una función que devuelva la suma total del saldo de todas las cuentas. 

Requisitos:

    • No tiene ningún parámetro de entrada
    • Devolverá una valor numérico de tipo monetario (DECIMAL(10,2) 

Actividad 2. MejorCliente

Crear una función que devuelva el id del mejor cliente. Siendo el mejor cliente aquél al que pertenece la cuenta de mayor saldo.

Actividad 3. Función SaldoCuentas

Se desea crear la función SaldoCuentas para la base de datos bancos, que me devuelva a partir de los movimientos (tabla movimientos) de una cuenta el saldo actual de dicha cuenta. Se trata de una procedimiento de verificación puesto que en la tabla cuentas tengo la columna saldo.

Observaciones:

    • Esta función recibirá como parámetro de entrada id de la cuenta.
    • Devolverá como parámetro de salida el saldo total verificado de dicha cuenta.

Actividad 4. Procedimiento VerificarSaldo

Crear un procedimiento para la base de datos Bancos llamado VerificarSaldo, este procedimiento comprobará que la columna saldo e la tabla Cuentas, coincide con el valor que devuelve la función desarrollada en el apartado anterior SaldoCuentas(). En caso de que no coincida deberá mostrar todos los detalles (todas las columnas) de esta cuenta.

El procedimiento recibirá como parámetro de entrada id de la cuenta que se quiere verificar.
